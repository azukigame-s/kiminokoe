# キミノコエ

失った記憶を、声で辿るアドベンチャー＆ノベルゲーム。

## シナリオ

### ルール

#### タイトル

`####` （h4） 記法で記載。

ただし、 `[]` でラッピングされているものはゲーム画面に表示しないタイトル。

#### サブタイトル

`#####` （h5）記法で記載。

ただし、 `[]` でラッピングされているものはゲーム画面に表示しないサブタイトル。

#### 本文

そのまま文章として記載。

1文字目の `　` （全角スペース）はゲーム画面に表示する必要はなし。

#### システムに関するもの

##### ページ送り

`> ⎘` で記載。

##### 背景変更

`> 🎨 [sea.jpg]` のように記載。

`[]` で囲まれたファイル名が背景として表示するファイル名。

##### エピソード呼び出し

`> 📞 [ep_0] 命日 を呼び出し` のように記載。

エピソード呼び出し後は元のシナリオに戻る。

エピソードはすべて回想になるので、エピソード表示中は背景をグレースケールにする。

##### シナリオ呼び出し

`> 📞 [shared_day_1010_c] 海水浴場 を呼び出し` のように記載。

エピソード呼び出しと同様に、呼び出し後は元のシナリオに戻る。

ただし、エピソード呼び出しと異なり、グレースケールにはならない。

シナリオ呼び出しの識別子は、対応する選択肢の識別子（例: `opt_day_1010_c`）の `opt` を `shared` に置換したものを使う。

##### シナリオジャンプ

`> ➡ [opt_day_1010_a] 「1. はい」を選択した場合 に進む` のように記載。

現在のシナリオから別のシナリオ（主に選択肢を選択したあとのシナリオ）にジャンプする。

ジャンプ後はもとのシナリオには戻らず、ジャンプ後のシナリオの進行に従う。

##### 選択肢

```
> [opt_day_1010_a]
> 
> 1. はい
> 2. いいえ
```

のように記載。

そのあとに、

> ```
> [opt_day_1010_a] 「1. はい」を選択した場合
> ```
>
> 「ふふふ、良かった。改めてお兄ちゃんおかえり！」
>
> ```
> [opt_day_1010_a] 「2. いいえ」を選択した場合
> ```
>
> 「ちょっとぉ、お兄ちゃんの冗談分かりづらいんだからやめてよね！」
>
> ```
> [opt_day_1010_a] 分岐終了
> ```

のように記載して、 if 文のようにシナリオを分割する。

ここに出てくる `[opt_day_1010_a]` が選択肢を一意にする目印で、他の選択肢との差別化を行う。

#### 設定

##### 地蔵焚

小さな罅が、大きな歪みを隠す村

###### 由来

主人公の故郷である地蔵焚（じぞうだき）。北に海、南に山と自然豊かなその村は、かつて古い『しきたり』を守っていた。

この村では兄弟それぞれが小さな地蔵を持つことにしていた。

兄弟が同時に病むと、どちらかの枕元に置いた地蔵を囲炉裏で焚べる。最初に罅の入った部分が「引き受けた病の場所」とされ、燃やしたその地蔵は川に流す。

燃やさなかった地蔵を持つ兄弟は必ず快復するが、引き受けた側は床に伏す。

やがて村人たちは気づいた——小さな地蔵を焚き続ければ、大きな災いを無数の小さな痛みに分けて払えることに。

この『しきたり』が根付き、村では毎晩、誰かが順番に小さな地蔵を火に焚べるようになった。

罅の入る音が聞こえるたび、村のどこかで誰かが小さく咳をし、指先を切り、階段で躓く。

燃やされた地蔵は山から流れる川へ——村人は「兄の水」「弟の水」と呼ぶ二筋の流れに、交互に流していく。

今ではそんな『しきたり』も忘れ去られているが、村の端には焚き場の跡が残っている。

地蔵を焚くことで災いを退ける……いつしかこの地は地蔵焚と呼ばれるようになった。

##### 童歌

祈りは忘れられ、調べだけが継ぐ

###### 童歌「ふたこじぞう」

じぞうさま　じぞうさま

どちらの子が　ピーヒョロロ

ひび割れた　ひび割れた

あたまおなか　手とあし

ぱちぱち燃えたら

川へゆく　川へゆく

###### 解説

この村では古くから童歌が残っている。 「ふたこじぞう」と呼ばれるこの歌は、村にある2体の並んだ地蔵にちなんだものだと思われているが、その由来は本来、過去の古い『しきたり』からくるものである。

今では子供たちも歌の意味を知らない。祖父母から教わり、ただ手遊びをしながら楽しげに口ずさむだけだ。

かつて毎晩誰かが小さな地蔵を焚べていた頃、この歌は村人たちにとって祈りであり、覚悟でもあった。

小さな痛みを引き受けることで、大きな災いから村を守る——その『しきたり』の記憶だけが、子守唄のように今も残っている。
