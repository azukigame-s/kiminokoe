# スグホラー演出 実装メモ

お節介編「日記の真っ黒ページ」シーンの専用演出。

---

## 演出シーケンス

| フェーズ | 内容 | 操作 |
|----------|------|------|
| クリック① | 「スグ。」が1行スタック | 1クリックで次へ |
| クリック② | 「スグ。」が2行目に積み重なる | 2クリックで次へ |
| クリック③ | 「スグ。」が3行目に積み重なる | 4クリックで次へ |
| 自動（単体）| 「スグ。」×6 が自動でスタック | 自動進行 |
| 自動（密集）| スグ×350=700文字の1行、黒文字、速度が徐々に加速 | 自動進行 |
| カオス | 700文字を背景に残したまま、画面全体にランダムな「スグ」が乱れ飛ぶ | 自動（操作不能） |
| 暗転 | カオスと同時進行でフェードブラック | 自動 |

**テキストの挙動:**
- クリック・自動フェーズのテキストは**消えずに積み重なる**（9行スタック）
- 通常のテキスト表示と同じ見た目（フォント・色・サイズ・インジケーター⏎）
- 密集フェーズ前にページクリア（プログラム的に自動）
- 密集行はページ折り返しありで黒文字表示、進行とともに速度加速
- 密集行の50%進行時点でカオスフォント（`FONT_PATH`）に一括切り替え
- 密集行の700文字はカオスフェーズ中も背景として残る（クリアしない）

## テスト方法

タイトルからお節介編まで進めなくても、演出単体でテストできる。

1. Godot エディタで `scenes/test/test_sugu_horror.tscn` を開く
2. **F6**（現在のシーンを実行）を押す
3. 演出が始まる。クリックまたは Enter / Space で進む

演出完了後はアウトプットに `[TEST] SuguHorrorDisplay 完了` と表示される。

## 調整パラメータ

`scripts/ui/sugu_horror_display.gd` の先頭にある定数を変更する。

| 定数 | 初期値 | 内容 |
|------|--------|------|
| `AUTO_INTERVAL` | `0.8` | 自動スグ1行の後の待機時間（秒） |
| `DENSE_SPEED_START` | `0.05` | 密集行アニメーション開始時の秒/文字（遅い） |
| `DENSE_SPEED_END` | `0.01` | 密集行アニメーション終了時の秒/文字（速い） |
| `DENSE_INTERVAL` | `1.0` | 密集行アニメーション完了後の待機時間（秒） |
| `CHAOS_INTERVAL_START` | `0.35` | カオス開始直後の生成間隔（秒） |
| `CHAOS_ACCEL` | `0.82` | 生成間隔の加速係数（小さいほど速く加速） |
| `CHAOS_INTERVAL_MIN` | `0.02` | 生成間隔の下限（秒） |
| `BLACKOUT_DURATION` | `8.0` | 暗転完了までの時間（秒） |
| `CHAOS_SIZE_MIN` | `30` | カオスラベルの最小フォントサイズ |
| `CHAOS_SIZE_MAX` | `300` | カオスラベルの最大フォントサイズ |

## フォントの差し替え方法

現在は `EnkaDotMincho24.ttf`（詩表示と同じフォント）を使っている。
怨霊フォントが手元に届いたら以下の手順で差し替える。

1. フォントファイルを `assets/fonts/` に置く（例: `onryou.ttf`）
2. `scripts/ui/sugu_horror_display.gd` の1行だけ変更する：

```gdscript
# 変更前
const FONT_PATH := "res://assets/fonts/EnkaDotMincho24.ttf"

# 変更後
const FONT_PATH := "res://assets/fonts/onryou.ttf"
```

このフォントは密集フェーズの50%進行時点とカオスラベルの両方に使われる。
