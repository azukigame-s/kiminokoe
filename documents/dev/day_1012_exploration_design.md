# 10月12日 探索パート 設計書

## 概要

10月12日の探索パートは、10日と同じ東西南北の4方向から行き先を選ぶ構造。
ただし **10日に訪問した場所は選択肢に表示されない**。
また、イロが同席しているため、10日と共用するシナリオは台詞・描写の調整が必要。

---

## フロー全体図

```
① おばあちゃんの問い
      │
      ├─ a「ゆっくりしてから帰る」
      │       └─ エピソード「怪魚」獲得 → 探索パートなし
      │
      └─ b「村を見て回ろうと思っている」
              └─ ② 探索前チェックへ

② 探索前チェック
      │
      ├─ cond_true_day1010 = true
      │       └─ エピソード「キャッチボール」獲得（未取得時のみ）
      │
      ├─ cond_true_day1011 = true（= キャッチボール取得済み）
      │       └─ トゥルーエンドルートへ（探索スキップ）
      │
      └─ cond_true_day1011 = false
              └─ ③ 探索選択へ

③ 探索選択
  ※ 10日に訪問済みの方角は選択肢に表示されない
      │
      ├─ 北（海の方）  → ④ 北ルート
      ├─ 東（湧き水の方）→ ⑤ 東ルート
      ├─ 南（お寺の方） → ⑥ 南ルート
      └─ 西（バス停の方）→ ⑦ 西ルート
```

---

## 各ルート詳細

### ④ 北ルート

- **バス停エピソード**（北・東共通）: 12日の探索でのみ発生。探索は1回限りのため重複しない
- 10日に北を選んでいない場合、10日と同じシーンを表示（バス停エピソード含む）
- **要検討**: イロが同席しているため、台詞・描写の調整が必要
- **要検討**: 10日との天気の違いを描写に反映

---

### ⑤ 東ルート

- 北ルートと同様
- **バス停エピソード**（北・東共通）: 12日の探索でのみ発生。探索は1回限りのため重複しない
- **要検討**: イロが同席しているため、台詞・描写の調整が必要
- **要検討**: 10日との天気の違いを描写に反映

---

### ⑥ 南ルート

選択肢が表示される。

| 選択肢 | 結果 |
|--------|------|
| a「お寺さんに会いに行く」 | エピソード「探検隊結成」獲得（JMRトリガー） |
| b「さらに南に進む」 | クワガタエピソードへ → ⑥-b |

#### ⑥-b クワガタエピソード（ループ選択）

道に迷う演出。「まっすぐ進む」「来た道を戻る」を繰り返し選べる。

**脱出条件（いずれか）:**

1. **正解シーケンス**: 戻る → 進む → 進む → 進む
2. **累計10回選択** で強制脱出

> **実装メモ**: 通常の `choice` + `jump` では対応が難しい。
> エンジンへの `set_flag` / `increment` / `branch_flag` コマンド追加が必要。

---

### ⑦ 西ルート

- 10日とは別物の新シナリオ
- エピソード「弟の水」獲得

---

## 必要なエンジン対応（実装時）

| 対応内容 | 用途 |
|----------|------|
| `choice` に `hidden_if` フィールド追加 | 10日訪問済み選択肢の非表示 |
| `set_flag` コマンド追加 | クワガタループのシーケンス追跡 |
| `increment` コマンド追加 | クワガタループの試行回数カウント |
| `branch_flag` / `branch_counter` コマンド追加 | 上記の条件分岐 |
