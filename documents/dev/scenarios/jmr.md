## 📋 概要

### タイムライン
- **1年前（メインストーリー）**: 主人公が声を失った状態で帰郷、スグの霊体が現れ、トゥルーエンドで成仏
- **現在（マルチシナリオ）**: 声を取り戻した1年後、再び帰郷

### コンセプト
「1年前の一連の出来事が、村のしきたりや童歌と偶然かつ奇跡的に重なっていたことに気づくカタルシス」

### 物語の構造
```
ホラー要素（強）
    ↓
謎の解明（真相への接近）
    ↓
理解と整理（論理的）
    ↓
感動のエンディング（ハッピーエンド）
```

### エンディングの核心
```
「スグ、お誕生日おめでとう」

やっと言える――12年間、言えなかった言葉。
```

## 📚 設定資料

### 地蔵焚のしきたり

#### 本来の仕組み（表向き・現在知られている）

1. 兄弟がそれぞれ小さな地蔵を持つ
2. 兄弟が同時に病むと、どちらかの地蔵を囲炉裏で焚べる
3. 最初に罅が入った部分が「引き受けた病の場所」とされる
4. 燃やした地蔵は川に流す
5. 燃やさなかった地蔵を持つ兄弟は必ず快復
6. 引き受けた側は床に伏す

#### 進化した使い方

- 毎晩誰かが順番に小さな地蔵を焚き続ける
- 大きな災いを無数の小さな痛みに分散
- 罅の入る音が聞こえるたび、村のどこかで誰かが小さく咳をし、指先を切り、階段で躓く
- 燃やされた地蔵は「兄の水」「弟の水」と呼ぶ二筋の流れに交互に流す

#### 10月との関係

- 10月は月が大きく美しく見える時期（十三夜など）
- 昔の人は大きな月を「不吉の兆し」と解釈
- 何かが満ちる＝災いが満ちると考えた
- だから10月に儀式を行い、災いを祓っていた

#### 現在

- しきたりは数十年前に途絶えた
- 焚き場の跡だけが残っている
- 童歌だけが残っている

### しきたりの真相（隠されている歴史）

#### 大昔の残酷な儀式

**兄弟のうち**:
- **片方は身代わりで死ぬ**（災いを引き受ける）
- **もう片方は口止めで喉を潰される**（秘密を守るため）

**結果**:
- 村は守られる
- でも兄弟は両方とも失われる
- 片方は命を、もう片方は声を

**後世の改変**:
- あまりに残酷なので「地蔵」に置き換えた
- 人ではなく地蔵を焼く
- 喉を潰す代わりに、地蔵に罅を入れる
- でも構造（型）は同じまま残った

**現在の状況**:
- 過去の犠牲者たちは成仏できていない
- 焚き場に魂が留まっている
- だから御札で鎮めている
- でも完全には鎮められていない

### 童歌「ふたこじぞう」- 二つのバージョン

#### イロが歌う「平和なバージョン」（現在広まっている）

```
じぞうさま　じぞうさま
どちらの子が　ピーヒョロロ
ひび割れた　ひび割れた
あたまおなか　手とあし
ぱちぱち燃えたら
川へゆく　川へゆく
```

**解釈（表向き）**:
- 地蔵様の歌
- 病を地蔵が引き受ける
- 焼いて川に流す
- 平和な民間信仰

---

#### おばあさんが知る「真実のバージョン」（大昔の歌詞）

```
じぞうさま　じぞうさま
どちらの子が　ピーヒョロロ
ひび割れた　ひび割れた
あたまおなか　のどくび
ぱちぱち焼いたら
川へやる　川へやる
```

**真実の意味**:

1. じぞうさま じぞうさま → 児蔵様（子供を蔵す）
2. どちらの子が ピーヒョロロ → どちらの子を選ぶか（強調）
3. ひび割れた ひび割れた → 喉が割れた（声帯が壊れた）
4. あたまおなか のどくび → 体の部位（特に喉）
5. ぱちぱち焼いたら → 遺体を焼く（人為的）
6. 川へやる 川へやる → 川に捨てる

**二つのバージョンの違い**:

| 部分 | イロ版 | おばあさん版 | 意味の違い |
|------|--------|------------|----------|
| 4行目 | 手とあし | **のどくび** | 手と足 vs 喉と首（重要！） |
| 5行目 | 燃えたら | 焼いたら     | 自然 vs 人為的             |
| 6行目 | 川へゆく | 川へやる     | 流れる vs 捨てる           |

**音はほぼ同じだが、意味が全く違う**

### 12年前の事故の真相

#### スグの事故は純粋な偶然

1. ~~スグが落としたキラカード（地蔵のカード）を拾おうとした~~
2. 車道に出てしまった
3. たまたま車が来た
4. 運が悪く、事故になった
5. **誰の悪意もない、純粋な事故**

#### 主人公が声を失ったのも

1. 弟の死のショック
2. 心因性失声症と診断された
3. **医学的に説明できる現象**

#### でも――「型」に当てはまった

1. たまたま10月だった
2. たまたま兄弟の片方が死に、片方が声を失った
3. たまたま古いしきたりの構造と一致した
4. **偶然が重なりすぎた**

**結果**:
- スグの魂が「型」に引っかかってしまった
- 成仏できなくなった
- 村の呪いではなく、構造的な問題
- **だから今は安全（主人公が型を完成させたので）**

### 1年前の出来事の真実

#### 主人公が無意識に童歌の手順を辿った

| 童歌の手順 | 主人公の行動（1年前） |
|-----------|-------------------|
| じぞうさま じぞうさま | ふたこじぞうを何度も見た（西・東・北ルート） |
| どちらの子が ピーヒョロロ | スグを想い続けた |
| ひび割れた ひび割れた | ep_0の記憶が「壊れた」→ ep_0_βへ修正 |
| あたまおなか のどくび | 主人公が引き受けた痛み（声＝喉） |
| ぱちぱち燃やしたら | 何かの「火」（比喩的な熱、感情） |
| 川へやる 川へやる | ep_6の兄の水、ep_2の海を訪れた |

**そして最後に**:
- 声を取り戻した
- 「スグ」と呼んだ
- 「型」が完成した
- だからスグは成仏できた

### 焚き場の御札の真相

**表向き**:
- 古い焚き場は危険だから立ち入り禁止
- 御札で封印している

**真相**:
- 過去何百年もの間、儀式で犠牲になった兄弟たちがいる
- 片方は殺され、もう片方は喉を潰された
- **両方とも成仏できていない**
- 焚き場にその魂が留まっている
- だから御札で鎮めている
- でも完全には鎮められていない

**御札の音「カラカラ」**:
- 表面的: 木の御札が風に揺れる音
- 真実: 喉を潰された犠牲者たちの「声なき声」
- 空っぽの喉から漏れる音
- 何かを訴えている

---

### 1体だけの地蔵の謎

**西ルート（センター跡）**:
- ふたこじぞうなのに1体だけ
- もう1体はどこに？

**真相**:
- ふたこじぞう = 兄弟を象徴
- 2体揃って初めて「成仏している」
- 1体だけ = **片方が成仏できていない**
- これは過去の犠牲者の地蔵
- もう1体は焚き場にある（魂が留まっている）

### 「偶然の奇跡」の核心

#### スグが主人公を導いた（かもしれない）

- 霊体として現れたのは、主人公に手順を教えるため
- 地蔵を見せ、思い出を蘇らせ、記憶を修正させた
- 最後に声を取り戻させた
- 全ては主人公を救うため

#### 主人公がスグを救った

- 無意識に儀式の手順を辿った
- 声を取り戻し、スグを呼んだ
- スグが成仏できた

#### お互いを救い合った = 兄弟の絆
