# 🏆 軌跡（トロフィー）

体験版では合計 **11個** の軌跡を取得できます。

---

## 通常の軌跡（7個）

各エピソードを回収すると取得できます。

| 軌跡名 | 説明文 |
|---|---|
| カード | 「キミと喧嘩をした日」 |
| 海 | 「波との戦い」 |
| バス停 | 「吹雪と兄弟」 |
| キャッチボール | 「下手だけど好きだから」※製品版限定 |
| 捨て猫 | 「シロとの出会い」 |
| 沢蟹 | 「このカニ食べれるの？」 |
| 神社 | 「実は高所恐怖症の兄」 |

<details>
<summary>💡 取得ヒントを見る</summary>

| 軌跡名 | ヒント |
|---|---|
| カード | 西ルートの先で… |
| 海 | 海水浴場で… |
| バス停 | 雨宿りをした場所で… |
| 捨て猫 | 用水路で… |
| 沢蟹 | 兄の水で… |
| 神社 | 鳥居の先で… |

</details>

## やりこみの軌跡（1個）

体験版限定の特別な軌跡です。

| 軌跡名 | 説明文 |
|---|---|
| 地蔵焚の旅人 | 「体験版でそこまでする？」 |

<details>
<summary>💡 取得ヒントを見る（ネタバレ注意）</summary>

すべてを巡り、耳を澄ませたとき。

</details>

<details>
<summary>⚠️ 取得条件を見る（ネタバレ大）</summary>

以下をすべて満たした状態で、童歌をスキップせずに最後まで聴くと取得できます。

1. プレイ時間3時間以上
2. 体験版の通常軌跡6個をすべて取得（キャッチボールを除く）
3. かくしの軌跡4個をすべて取得

</details>
