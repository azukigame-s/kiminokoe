## キミノコエ -体験版-
失った記憶を、声で辿るノベルゲーム「キミノコエ」の体験版です。

### 📜 このゲームについて

12年前、弟は事故で亡くなった。

故郷「地蔵焚」に帰省した主人公を待っていたのは、
懐かしい景色と、曖昧になった記憶。

「なんで喧嘩なんてしてしまったんだろう……」

思い出せない。

村を歩き、少しずつ記憶を取り戻していく中で、
主人公は白い服の少年と出会う。

弟に似た、その少年は――。

選択によって変わる物語。
周回プレイで明かされる、隠された真実。

そして最後に――
キミノコエは、聞こえる。

**■ ジャンル**
サウンドノベル / 感動 / ミステリー

**■ ゲームシステム**

- **自由探索**: 村を自由に歩き、思い出を集める
- **マルチエンディング**: 選択と回収状況で結末が変化
- **周回プレイ**: 1周では到達できないトゥルーエンド
- **トロフィーシステム**: 取りこぼしを確認しながら周回

### 📖 収録内容
