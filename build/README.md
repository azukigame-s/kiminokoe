## キミノコエ -体験版-
失った記憶を、声で辿るノベルゲーム「キミノコエ」の体験版です。

### 📜 このゲームについて

12年前、弟は事故で亡くなった。

故郷「地蔵焚」に帰省した主人公を待っていたのは、
懐かしい景色と、曖昧になった記憶。

「なんで喧嘩なんてしてしまったんだろう……」

思い出せない。

村を歩き、少しずつ記憶を取り戻していく中で、
主人公は白い服の少年と出会う。

弟に似た、その少年は――。

選択によって変わる物語。
周回プレイで明かされる、隠された真実。

そして最後に――
キミノコエは、聞こえる。

**■ ジャンル**
サウンドノベル / 感動 / ミステリー

**■ ゲームシステム**

- **自由探索**: 村を自由に歩き、思い出を集める
- **マルチエンディング**: 選択と回収状況で結末が変化
- **周回プレイ**: 1周では到達できないトゥルーエンド
- **トロフィーシステム**: 取りこぼしを確認しながら周回

### 📖 収録内容
- １０月１０日（ゲーム内日時）までの全シナリオ
- エピソード６本
- 軌跡（トロフィー）システム：体験版内で１１個取得可能

### 💻 動作環境
- Windows 11
- macOS

### 🎮 プレイ方法

#### Windows
1. Assets 内の `kiminokoe_demo_v0.5.4_windows.exe.zip` をダウンロードする
2. zip ファイルを解凍する
3. `kiminokoe_demo_v0.5.4_windows.exe` を実行する

#### macOS
1. Assets 内の `kiminokoe_demo_v0.5.4_macos.zip` をダウンロードする
2. zip ファイルを解凍する
3. `キミノコエ体験版.app` を起動する

> **初回起動時の注意（macOS）**
>
> セキュリティ設定により「開発元を確認できない」とブロックされる場合があります。
> その際は以下の手順で開いてください。
>
> 1. Apple メニュー → 「システム設定」を開く
> 2. 左サイドバーの「プライバシーとセキュリティ」を選択
> 3. 「セキュリティ」セクションに表示された「このまま開く」をクリック
> 4. パスワードまたは Touch ID で認証 → 「開く」を選択
>
> 以降は通常通り起動できます。
